# TBADAO_Aragon_test1

This repository is meant as a test repository for the test-DAO "TBADAOTest1" running on Aragon. The objective consisist in determining how much this repository can serve as the operational platform for the DAO. The DAO serves as a decentralised collaborative platform allowing Blockchain consultants to work together on Blockchain consulting projects.
